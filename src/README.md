### [分治法](/Algorithm/DivideAndConquer)

> 发布于 2026-07-09

分治法是一种算法设计策略，其核心思想是将原问题分解为若干个规模较小的子问题，递归地解决各个子问题，最后将子问题的解合并为原问题的解。
