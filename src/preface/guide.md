# 使用指南

## 如何阅读

本博客使用 VuePress 构建，你可以通过顶部导航栏浏览不同分类的文章。

## 本地运行

如果你想本地运行博客：

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建静态文件
pnpm build
```

## 文章结构

- `src/` - 所有文章内容都放在这个目录下
- `src/preface/` - 前言相关
- `src/Algorithm/` - 算法学习相关
